# Jest Test

#### Trying out Jest.

The purpose of this project is to practice testing in JavaScript usig Jest

The following functions are created and tested:

 - capitalize(string)
 - reverseString(string)
 - caesarCipher
 - calculator(add, substract, divide, multiply)
